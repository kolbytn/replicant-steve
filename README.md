# Replicant Steve

Steve's a Replicant!

## Installation

Install Node.js >= 14 from [nodejs.org](https://nodejs.org/)

Then, install mineflayer:
```
npm install mineflayer
```

Install python packages:
```
pip install -r requirements.txt
```

## Usage

Start minecraft server on port `55916`

run `python main.py`
